# docker-setup
Template docker yaml files for quick reference.

## Contents of interest:
* azurite
* kafka
* zookeeper
* mongo
* redis
* redis-commander

## Notable Commands

*Start Docker*
```console
docker-compose -f <DOCKER_CONFIG_FILENAME> up -d
```

*Stop Docker*
```console
docker-compose -f <DOCKER_CONFIG_FILENAME> down
```
